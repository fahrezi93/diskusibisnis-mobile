# Panduan Menjalankan DiskusiBisnis Mobile (Flutter)

Aplikasi ini menggunakan Framework Flutter.

## Persiapan
1. Pastikan Anda telah menginstal **Flutter SDK**.
2. Buka terminal di folder `mobile`.
3. Jalankan `flutter pub get` untuk mengunduh library yang dibutuhkan.

## Menjalankan Aplikasi
1. Sambungkan HP Android/iOS atau jalankan Emulator.
2. Ketik perintah:
   ```bash
   flutter run
   ```

## Struktur Project
- `lib/main.dart`: Titik masuk aplikasi.
- `lib/screens/`: Berisi halaman-halaman utama (Home, Profil, dll).
- `lib/theme/`: Konfigurasi warna (Emerald Green).

Selamat coding!
